# HSDöner

## Über das Projekt

**HSDöner** ist ein kooperatives Multiplayer-Spiel, das im Rahmen eines Informatikprojekts (Einführung in die Spielentwicklung mit Godot) entwickelt wurde. Das Spiel ist inspiriert von Overcooked und versetzt die Spieler in die Rolle von Dönerladen-Mitarbeitern, die gemeinsam Bestellungen zubereiten und ausliefern müssen.

## Features

- Lokaler Multiplayer für 2 Spieler
- Verschiedene Zutaten und Rezepte
- Zeitbasierte Herausforderungen
- Kooperative Aufgabenverteilung

## Installation

1. Über [Godot Engine](https://godotengine.org/) v4.5.1 installieren 
2. Repository klonen
3. Projekt in Godot öffnen und starten
